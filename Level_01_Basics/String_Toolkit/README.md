#  String Toolkit

## Description

String Toolkit is a Flask web application that analyzes and transforms user-entered text using Python string methods.

The application demonstrates various string manipulation techniques and presents the results through an interactive web interface.

## Features

- Convert text to Uppercase
- Convert text to Lowercase
- Convert text to Title Case
- Character Count
- Word Count
- Reverse Text
- Responsive Card-Based Interface

## Technologies Used

- Python
- Flask
- HTML5
- CSS3

## Concepts Practiced

- String Methods
- String Slicing
- len()
- split()
- Dictionaries
- Flask Forms
- Dynamic Rendering
