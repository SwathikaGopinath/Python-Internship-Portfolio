# 🔍 Data Type Explorer

## Description

Data Type Explorer is a Flask application that demonstrates Python's built-in data types by displaying both user-entered values and their corresponding data types.

This project helped strengthen my understanding of Python variables and dynamic rendering using Flask.

## Features

- Accepts different user inputs
- Displays value and data type
- Clean table layout
- Dynamic Flask rendering

## Technologies Used

- Python
- Flask
- HTML5
- CSS3

## Concepts Practiced

- Variables
- Python Data Types
- type()
- Dictionaries
- Loops
- Flask Templates
- Jinja2
