# Simple Calculator

## Description

Simple Calculator is a Flask web application that performs basic arithmetic operations on two user-provided numbers.

The application demonstrates backend processing, form handling, and exception handling using Flask.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Floor Division
- Modulus
- Power
- Division by Zero Handling
- Responsive User Interface

## Technologies Used

- Python
- Flask
- HTML5
- CSS3

## Concepts Practiced

- Arithmetic Operators
- Exception Handling
- Dictionaries
- Flask Forms
- GET & POST Requests
- Dynamic HTML Rendering
