# Hello Card

## Description

Hello Card is a beginner-friendly Flask web application that collects a user's name and age through a web form and displays a personalized greeting card.

This project was developed as the first project in my Python Internship Portfolio to understand the basics of Python, Flask, HTML, and CSS integration.

## Features

- Accepts user name and age
- Displays a personalized greeting
- Calculates next year's age
- Responsive web interface
- Clean and simple UI

## Technologies Used

- Python
- Flask
- HTML5
- CSS3

## Concepts Practiced

- Variables
- User Input
- Data Type Conversion
- f-Strings
- Flask Routing
- GET & POST Requests
- HTML Forms
- CSS Styling
