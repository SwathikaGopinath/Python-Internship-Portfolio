# Shopping Cart Price Analyzer

## Overview

Shopping Cart Price Analyzer is a Flask web application that allows users to add shopping items along with their prices, displays the complete shopping cart, and identifies the most expensive and the least expensive item.

This project was developed as part of the Python Internship Portfolio.

---

## Features

- Add shopping items dynamically
- Enter item names and prices
- Display all items in a structured table
- Identify the most expensive item
- Identify the least expensive item
- Responsive user interface using HTML and CSS
- Flask backend with Jinja2 templating

---

- Flask
- HTML5
- CSS3

## Installation

Clone the repository.

```bash
git clone <repository-url>
```

Install the required dependencies.

```bash
pip install -r requirements.txt
```

Run the application.

```bash
python app.py
```

Open your browser and navigate to:

```
http://127.0.0.1:5000/
```
