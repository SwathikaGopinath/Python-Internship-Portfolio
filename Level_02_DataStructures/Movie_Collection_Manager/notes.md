# Movie Collection Manager - Notes

## Objective

Develop a Flask-based web application to manage a movie collection using Python lists.

- Functions
- Conditional Statements
- append()
- remove()

- Routes
- render_template()

### HTML & CSS
- Forms
- Jinja Templates
- Flexbox
- Responsive Design
- Hover Effects

## Routes

| Route | Method | Description |
|--------|--------|-------------|
| / | GET | Home page |
| /add | POST | Add a movie |
| /sort | GET | Sort movies |
| /delete/<movie> | GET | Delete a movie |
| /clear | GET | Clear all movies |

## Project Flow

```
User Input
     ↓
Flask Route
     ↓
Python List Operation
     ↓
Updated Movie List
     ↓
Rendered HTML Page
```
