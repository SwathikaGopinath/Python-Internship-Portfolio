# Unique Words Finder

## Overview

Unique Words Finder is a Flask-based web application that accepts a sentence from the user, identifies all unique words using Python's `set` data structure, and calculates the number of duplicate words removed.

This project was developed as part of the Python Internship Portfolio.

---

## Features

- Accepts a sentence from the user
- Splits the sentence into individual words
- Identifies unique words using a set
- Displays the number of duplicate words removed
- Responsive user interface using HTML and CSS
- Flask backend with Jinja2 templating

## Installation

Clone the repository.

```bash
git clone <repository-url>
```

Install the required dependencies.

```bash
pip install -r requirements.txt
```

Run the application.

```bash
python app.py
```

Open your browser and navigate to:

```
http://127.0.0.1:5000/
```
